# DHBW
DHBW Labs - Web Programmierung

## Build Lab
Open a CMD shell on Windows. Goto directory lab.x and call
<pre>
.\buildServer.bat
</pre>

## Run Lab
Open a CMD shell on Windows. Goto directory lab.x and call
<pre>
.\startServer.bat
</pre>

## Test Lab
Open a browser and locate
<pre>
http://localhost:8080/
</pre>
